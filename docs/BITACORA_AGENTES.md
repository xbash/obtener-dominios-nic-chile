# Bitácora de agentes

## Propósito
Registrar intervenciones relevantes realizadas con agentes LLM/Codex/ChatGPT para mantener trazabilidad del proyecto.

## Regla de uso
Cada vez que un agente cree, modifique o continúe este proyecto, debe registrar una entrada breve. No debe inventar versión/modelo. Si no puede verificarlo, debe indicar `pendiente-de-verificación`.
