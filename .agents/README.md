# Agentes locales del proyecto

Este directorio contiene adaptadores locales para usar skills reutilizables en el contexto de `obtener-dominios-nic-chile`.

## Estructura activa

- `skills/ingenieria-software/`: adaptacion local de ingenieria de software.
- `skills/seguridad-appsec/`: adaptacion local de seguridad aplicativa.

## Fuentes reutilizables

Las skills completas viven fuera del repositorio, en la libreria local de
skills reutilizables del mantenedor:

- `skills/ingenieria-software`
- `skills/seguridad-appsec`

No duplicar aqui las skills completas salvo que se necesite una copia congelada por auditoria.

## Copias archivadas

`_copias-reutilizables-no-usar/` contiene copias identicas detectadas el 2026-07-20. No deben usarse como fuente activa; estan solo para no borrar trabajo sin revision manual.
